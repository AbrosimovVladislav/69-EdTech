# Структура продукта

## Принцип

Один спринт — один формат — один результат. Сначала валидируем один сильный спринт, потом второй, потом расширяем линейку. Не наоборот.

Формат спринта единый для любой ниши. Меняется только содержание: задачи, шаблоны, примеры, промпты. Ритм, механика, структура недель — одинаковые. Это делает продукт повторяемым и масштабируемым: адаптация под новую профессию — 2-3 недели, а не месяцы.

---

## Формат спринта

- **Длительность:** 3-4 недели
- **Группа:** 12-15 человек (B2C) или 5-10 (B2B-команда)
- **Цена:** 15-30К руб (B2C), x2-x3 за B2B-пакет
- **Старт:** фиксированная дата, когортное прохождение
- **Стек:** Zoom + Telegram + Notion

Когортный формат — не случайный выбор. Completion rate у когорт: 80-96%. У self-paced: 3-20%. Давление группы, дедлайнов и живых разборов — причина, по которой люди доходят до конца и получают результат.

---

## Еженедельный ритм

Каждая неделя спринта устроена одинаково:

1. **Live-сессия** (60-90 мин) — разбор темы на реальных задачах, демонстрация, Q&A. Не лекция, а рабочая сессия.
2. **Практическое задание** — внедрение в свою работу, не учебный кейс. Человек делает то, что будет использовать дальше.
3. **Разбор в группе** — Telegram-чат + письменный фидбек. Видишь, как другие решают похожие задачи.
4. **Шаблоны и промпты** — готовые, copy-paste, заточенные под задачи конкретной профессии.

---

## Структура недель

**Неделя 1: Первый результат.**
Минимум теории. Выбор инструмента, анатомия промпта — ровно столько, сколько нужно для старта. Главное — первый рабочий процесс с AI, который даёт ощутимый результат к концу недели. Quick win, который показывает: это работает.

**Неделя 2: Ключевые рабочие задачи.**
Глубокое погружение в 2-3 задачи, на которые уходит больше всего времени. Шаблоны, промпты, пошаговые процессы. Задание: переделать реальный рабочий процесс с AI и замерить разницу «до и после».

**Неделя 3: Аналитика и данные.**
AI для задач, которые обычно откладываются: анализ данных, отчёты, ресёрч. Работа с CSV, таблицами, метриками. AI не как генератор текста, а как аналитик.

**Неделя 4 (опционально): Система.**
Цепочки промптов, кастомные ассистенты, автоматизация рутины. Выпускной проект: рабочий AI-workflow, который можно использовать с понедельника.

---

## Что получает участник

- 2-3 настроенных AI-workflow для своих реальных задач
- Библиотека промптов под профессию (copy-paste ready)
- Конспекты live-сессий с примерами
- Замер «до и после»: насколько изменилось время на типовые задачи
- Навыки, которые переносятся на любой инструмент: декомпозиция задач, системный промптинг, оценка результатов AI, работа с данными

Ключевое: человек уходит не с «пониманием AI», а с работающей системой, которую уже начал использовать во время спринта.

## Адаптация под первую волну

### Спринт: AI для аналитика данных

**Целевой результат:** система AI-workflows для обработки данных, построения отчётов и ресёрча — автоматизация рутинных аналитических задач.

**Неделя 1 — AI для работы с данными: первый workflow.** Загрузка и предобработка данных с AI. Генерация SQL-запросов, формул, скриптов по описанию задачи. Быстрый EDA (exploratory data analysis) через AI. *Задание:* взять реальный рабочий датасет, построить базовый анализ с AI, замерить время.

**Неделя 2 — Отчёты, дашборды, визуализация.** AI для генерации отчётов из сырых данных. Автоматизация повторяющихся расчётов. Подготовка выводов и рекомендаций. *Задание:* переделать реальный рабочий отчёт с AI и сравнить с ручным процессом.

**Неделя 3 — Ресёрч и система.** AI для поиска и синтеза информации, конкурентного анализа, подготовки обзоров. Цепочки промптов для типовых задач. Кастомные ассистенты. *Выпускной:* рабочий AI-workflow для своих аналитических задач.

**Модель:** B2C. Каналы: Telegram-сообщества аналитиков, профчаты, партнёрские вебинары.

---

### Спринт: AI для HR / рекрутинга

**Целевой результат:** AI-система для подбора — ускорение скрининга, подготовки к интервью и HR-отчётности.

**Неделя 1 — AI в подборе.** Скрининг резюме с AI, генерация описаний вакансий, sourcing кандидатов. Саммари интервью. *Задание:* обработать пачку резюме через AI-workflow, сравнить скорость и качество с ручным скринингом.

**Неделя 2 — HR-аналитика и документация.** Анализ данных по воронке подбора, time-to-hire, причинам отказов. Автоматизация отчётности. Подготовка HR-документов. *Задание:* собрать аналитический отчёт по реальным данным.

**Неделя 3 — Система и автоматизация.** Полный workflow подбора с AI. Кастомные ассистенты для HR-задач. *Выпускной:* AI-workflow для своего процесса подбора.

**Модель:** B2C + B2B (спринт для HR-отдела). Каналы: outreach на HRD, HR-сообщества, профчаты.

---

### Спринт: AI для веб-/продуктового дизайнера

**Целевой результат:** AI-workflows для процесса дизайна — от генерации идей и референсов до готовых макетов и ассетов.

**Неделя 1 — AI в дизайн-процессе: от идеи к макету.** Генерация концепций, мудбордов и референсов с AI. AI для UI-компонентов и layout. Работа с генерацией изображений для лендингов и продуктов. *Задание:* переделать реальную дизайн-задачу с AI, замерить время.

**Неделя 2 — AI для рутины и ассетов.** Генерация иконок, иллюстраций, фонов. Адаптация макетов под форматы. Копирайтинг для UI. Подготовка ассетов для разработки. *Задание:* собрать набор ассетов для реального проекта с AI.

**Неделя 3 — Workflows и система.** Цепочки для полного дизайн-процесса. Кастомные ассистенты для типовых задач. Интеграция AI в Figma-процесс. *Выпускной:* AI-workflow для своего дизайн-процесса.

**Модель:** B2C. Каналы: дизайн-сообщества, Telegram, Behance/Dribbble-комьюнити, партнёрские вебинары.

## Следующие волны

Когда первые спринты стабильно набирают когорты и фреймворк подтверждён — расширяем линейку. Формат тот же, содержание другое, для каждого нового сегмента привлекаем доменную экспертизу по необходимости.

**Вторая волна:** маркетологи (контент, аналитика ЦА, ресёрч), sales (outreach, CRM-аналитика, персонализация), PM-ы.

**Позже:** юристы (проверка договоров, правовой ресёрч), финансисты, предприниматели.

---

## Что дальше (целевая архитектура)

Спринт — ядро продукта. Когда 2-3 спринта стабильно работают, вокруг них выстраивается воронка:

**Бесплатный контент** (YouTube, Telegram) — одно видео = одна задача с AI. Роль: привлечение и доверие.

**Self-paced** (9-19 EUR) — структурированная версия материалов спринта. Rolling start. Роль: фильтр для когорт, вечнозелёный доход.

**Расширенная когорта** (300-800 EUR) — 6-8 недель, глубже программа, живые менторы. Upsell для выпускников спринта.

Но это — не план на день один. Это план на после того, как спринт валидирован.
