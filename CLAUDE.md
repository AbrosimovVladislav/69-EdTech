# EdTech AI Skills — Проект образовательного бизнеса

## О проекте

Образовательный бизнес в сфере AI-навыков. Профессиональные и общие AI-навыки для разных категорий людей (PM, HR, маркетологи, финансисты, юристы и др.).

Бизнес-модель: трёхуровневая воронка:
1. **YouTube** (бесплатно) — привлечение аудитории, доверие, email-база
2. **Self-paced курс** (€9–19) — монетизация широкой аудитории, фильтр для когорт
3. **Когортный курс / AI-менторство** (€200–500) — основной генератор прибыли (72–81% маржа)

Базовый документ с обзором рынка и моделью: `edtech_opportunity_deck.md`

## Структура проекта

- `edtech_opportunity_deck.md` — обзор возможностей и модель продукта
- `docs/` — вся проектная документация
- `docs/INDEX.md` — оглавление документации (читать первым)

## Как работать с проектом

1. При старте новой сессии — читай этот файл и `docs/INDEX.md`
2. INDEX.md содержит оглавление всех документов с описаниями и статусами
3. Все исследования, решения и планы фиксируются в docs/
4. При изменении документации — обновляй INDEX.md

## Язык

Основной язык проекта и документации — русский.
