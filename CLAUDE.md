# EdTech AI Skills — Проект образовательного бизнеса

## О проекте

Каталог прикладных AI-спринтов для профессионалов. Каждый спринт — 3-4 недели, одна профессия, один измеримый результат. Первая волна: маркетологи + sales. Вторая: HR + юристы.

Целевая архитектура (после валидации спринтов): YouTube (бесплатно) → Self-paced (9-19 EUR) → Расширенная когорта (300-800 EUR) → Alumni-community.

## Структура проекта

- `README.md` — описание проекта и оглавление
- `edtech_opportunity_deck.md` — обзор рынка и модель продукта
- `docs/` — рабочая документация (оглавление: `docs/README.md`)
- `claude_request.md` — шаблон запроса на исследование

## Как работать с проектом

1. При старте сессии — читай этот файл и `README.md`
2. `docs/README.md` содержит оглавление всех документов с описаниями и статусами
3. Все исследования, решения и планы фиксируются в docs/
4. При изменении документации — обновляй `docs/README.md` и `README.md`
