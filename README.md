# EdTech AI Skills

Каталог прикладных AI-спринтов для профессионалов. Каждый спринт — 3-4 недели интенсивной работы, где специалист конкретной профессии внедряет AI в свои реальные рабочие процессы и уходит с готовой системой.

Принцип: **один спринт = одна профессия = один измеримый результат.**

## Статус проекта

Стадия: подготовка к первому запуску. Первая волна — маркетологи (B2C) + sales (B2C + B2B-тест).
